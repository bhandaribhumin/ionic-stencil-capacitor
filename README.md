# ionic4-stencil-capacitor
this project contains ionic, stencil and capacitor setup !REF https://github.com/ionic-team/ionic-pwa-toolkit
```bash
npm install
npm run build
npx cap add ios
 cd ios/App
 pod install
 cd .. / ..
npx cap copy
npm run-script build-cap
```

Open ios/App/App.xcworkspace in Xcode
